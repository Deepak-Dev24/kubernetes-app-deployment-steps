Command Reference for React App Deployment on Kubernetes with Docker and Minikube

Command Description
git clone https://github.com/Deepak-Dev24/react.git Clones the react repository from GitHub to your local machine.
cd react Navigates to the react project directory.
npm start Starts the React development server.
npm install Installs the required dependencies for the React project.
npm start Starts the React development server (again after installing dependencies).
docker login Logs into Docker Hub using your credentials.
docker info Displays detailed information about your Docker installation and configuration.
docker build -t deepak2633/new-react-app:v1 . Builds a Docker image for the React app and tags it as deepak2633/new-react-app:v1.
docker images Lists all available Docker images on your system.
docker push deepak2633/new-react-app:v1 . Pushed the new-react-app:v1 Docker image to Docker Hub.
docker push deepak2633/new-react-app:v1 (Repeated) Pushes the new-react-app:v1 Docker image to Docker Hub.
minikube status Checks the status of the Minikube cluster.
minikube start Starts the Minikube virtual machine.
kubectl create deployment mynew-app --image=deepak2633/webapp-demo:01 Creates a new deployment using the webapp-demo:01 Docker image.
kubectl create deployment mywebapp --image=deepak2633/webapp-demo:01 Creates a deployment named mywebapp using the webapp-demo:01 Docker image.
kubectl get deployment Lists all deployments in the Kubernetes cluster.
kubectl get pods Lists all pods running in the cluster.
kubectl describe pod mynew-app-67bdbc8f69-mpzhs Provides detailed information about a specific pod to help with debugging.
minikube dashboard Launches the Minikube Dashboard for visual management of your Kubernetes resources.
minikube dashboard --url Provides the URL to access the Minikube dashboard in a browser.
kubectl logs mywebapp-5d469d79d6-p8dsg Retrieves logs from the specified pod (mywebapp pod).
kubectl expose deployment mywebapp --type=NodePort --port=3000 Exposes the mywebapp deployment to be accessible externally on port 3000 via NodePort.
kubectl delete svc my-pod-service Deletes a service named my-pod-service from the Kubernetes cluster.
kubectl expose pod <pod-name> --type=NodePort --port=80 --target-port=8080 --name=my-pod-service Exposes a specific pod to be accessible externally through a NodePort with specified port mappings.
kubectl set image deployment mywebapp webapp-demo=deepak2633/webapp-demo:06 Updates the mywebapp deployment to use the webapp-demo:06 Docker image.
kubectl create deployment my-react-app --image=deepak2633/new-react-app:v1 Creates a deployment for your React app using the new-react-app:v1 Docker image.
kubectl expose deployment my-react-app --type=NodePort --port=80 --target-port=3000 Exposes the React app (my-react-app) on port 80 externally, with the internal target port 3000.
kubectl get services Lists all services in the cluster, including NodePort services.
